### Steps to run GDC hardware API wrapper
```
python3 -m venv venv

Activate virtual environment:

windows: .\venv\Scripts\activate
mac: source venv/bin/activate

pip install -r requirements.txt
``` 

Test:
```
python -m unittest discover -s tests
```

Run:
```
python src/main.py
```
